## API

### Popover Props

| 名称      | 类型                                     | 默认值  | 说明              |
| --------- | ---------------------------------------- | ------- | ----------------- |
| to        | `string`                                 | `body`  | 弹出内容的位置    |
| placement | `string`                                 | `top`   | popover的弹出位置 |
| trigger   | `hover \| click \| focus \| contextmenu` | `hover` | 触发方式          |
| disabled  | `boolean`                                | `false` | 是否禁用          |
