## API

### Anchor Props

| 名称       | 类型                 | 默认值  | 说明                 |
| ---------- | -------------------- | ------- | -------------------- |
| affix      | `boolean`            | `false` | 是否启用固钉         |
| affixProps | `object`             | ``      | 启用固钉时的固钉属性 |
| bounds     | `number`             | `10`    | 锚点区域边界         |
| container  | `string \| function` | ``      | 指定的滚动容器       |

### AnchorLink Props

| 名称  | 类型     | 默认值 | 说明     |
| ----- | -------- | ------ | -------- |
| href  | `string` | ``     | 锚点链接 |
| title | `string` | ``     | 锚点文本 |
