## API

### Backtop Props

| 名称             | 类型     | 默认值 | 说明                 |
| ---------------- | -------- | ------ | -------------------- |
| visibilityHeight | `number` | `150`  | 滚动达此值，按钮出现 |
| right            | `number` | `40`   | 距离页面右边距离     |
| bottom           | `number` | `40`   | 距离页面底部距离     |
| target           | `string` | ``     | 触发滚动的目标       |
