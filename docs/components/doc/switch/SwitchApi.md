## API

### Switch Props

| 名称                | 类型      | 默认值                  | 说明           |
| ------------------- | --------- | ----------------------- | -------------- |
| modelValue\/v-model | `boolean` | `false`                 | 开关受控下的值 |
| color               | `string`  | `rgba(42, 148, 125, 1)` | 背景颜色       |
| round               | `boolean` | `true`                  | 是否为圆形     |
| disabled            | `boolean` | `false`                 | 是否禁用       |
