## API

### Card Props

| 名称      | 类型      | 默认值      | 说明             |
| --------- | --------- | ----------- | ---------------- |
| bordered  | `boolean` | `true`      | 是否显示卡片边框 |
| hoverable | `boolean` | `false`     | 卡片是否可悬浮   |
| title     | `string`  | `undefined` | 卡片标题         |

### Card Slots

| 名称    |     |     | 说明         |
| ------- | --- | --- | ------------ |
| header  |     |     | 头部内容     |
| extra   |     |     | 头部额外内容 |
| default |     |     | 卡片内容     |
| footer  |     |     | 底部内容     |
| action  |     |     | 操作区域内容 |
