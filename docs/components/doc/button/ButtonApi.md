## API

### Button Props

| 名称     | 类型                                                | 默认值    | 说明             |
| -------- | --------------------------------------------------- | --------- | ---------------- |
| type     | `default \| primary \| success \| warning \| error` | `default` | 按钮的类型       |
| round    | `boolean`                                           | `false`   | 按钮是否显示圆角 |
| disabled | `boolean`                                           | `false`   | 按钮是否禁用     |
