## API

### Input Props

| 名称                | 类型                                  | 默认值      | 说明         |
| ------------------- | ------------------------------------- | ----------- | ------------ |
| modelValue\/v-model | `string \| number`                    | `undefined` | 输入框的值   |
| type                | `text \| textarea \| password \| ...` | `text`      | 输入框类型   |
| round               | `boolean`                             | `false`     | 是否显示圆角 |
| placeholder         | `string`                              | `undefined` | 输入框占位符 |
| state               | `default \| error \| warning `        | `default`   | 输入框状态   |
