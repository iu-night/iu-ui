## API

### Checkbox Props

| 名称 | 类型 | 默认值 | 说明 | 
| --- | --- | --- | --- | 
| modelValue\/v-model | `boolean` | `false` | 复选框的勾选状态 |
| color | `string` | `rgba(42, 148, 125, 1)` | 背景颜色 |
| checkColor | `string` | `#fff` | 勾选颜色 |
| round | `boolean` | `false` | 是否为圆形 |
| disabled | `boolean` | `false` | 是否禁用 |
