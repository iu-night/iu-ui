## API

### Modal Props

| 名称 | 类型 | 默认值 | 说明 | 
| --- | --- | --- | --- | 
| show | `boolean` | `false` | 是否显示模态框 |
| title | `string` | `undefined` | 模态框标题 |
| maskClose | `boolean` | `true` | 点击遮罩时是否关闭模态框 |

| footer | | | 底部内容 | 
| action | | | 操作区域内容 |
